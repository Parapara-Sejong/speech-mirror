# 가격 카드 CTA 버튼 위치 통일

## 개요
`PricingTierCard` 컴포넌트의 CTA 버튼을 가격 바로 아래로 이동하고, 기능 목록을 구분선과 함께 하단으로 분리했다. 티어별 기능 개수가 달라도 카드 간 CTA 버튼 위치가 일정하게 정렬되도록 통일성을 확보했다.

## 변경 파일
- `frontend/src/components/ui/PricingTierCard.tsx`

## 변경 내용
- 레이아웃 순서 변경: `이름 → 가격 → CTA 버튼 → 구분선 → 기능 목록`
  - 기존: `이름 → 가격 → 기능 목록 → CTA 버튼(하단)`
- 기능 목록 상단에 구분선(`border-t`) + 상단 여백(`pt-8`) 추가
- 구분선 색상 분기
  - featured(dark surface): `border-on-dark/10`
  - 일반(cream surface): `border-hairline`

## 효과
- 기능 개수(무료 3 / 프로 4 / 맥스 3)와 무관하게 CTA 버튼이 동일한 높이에 위치
- 레퍼런스 디자인과 동일한 카드 구조로 시각적 통일성 확보

## 검증
- `npm run build` 통과 (tsc + vite, 에러 0)
