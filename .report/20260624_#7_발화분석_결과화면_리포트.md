# 발화 분석 결과 화면(단일 답변 리포트) 구현

### 📌 작업 개요

PRD §11 결과 리포트(단일 답변 1개)를 보여주는 `/result` 페이지 신설. 백엔드 없이 임시(mock) 데이터로 단독 동작·시연 가능. 상단 고정 오디오 플레이어 + 발화/침묵/필러 타임라인을 중심으로, 전사·점수·말속도·침묵·필러·내용 피드백·개선 포인트를 한 화면에 표시.

보고서 파일: `.report/20260624_#7_발화분석_결과화면_리포트.md`

### 🎯 구현 목표

- 결과 데이터 계약(PRD §11) 타입화 + 임시 fixture로 백엔드 없이 렌더
- 다시 들으며 분석 확인: 재생 위치 ↔ 타임라인 구간 ↔ 전사 동시 표시
- 디자인 시스템(cream/coral/dark) 토큰으로 일관된 결과 화면
- 점수대별 반응 마스코트로 피드백 톤 강화

### ✅ 구현 내용

#### 데이터 계약·임시 데이터·조회 훅
- 파일: `frontend/src/features/analysis/types.ts`, `mockReport.ts`, `useAnalysisReportQuery.ts`
- 변경 내용: `AnalysisReport`(질문·평가역량·종합/세부 점수·전사·오디오·음성지표·타임라인·내용 피드백·개선 포인트) 타입 추가. `MOCK_REPORT` fixture를 TanStack Query `queryFn`이 짧은 지연 후 반환
- 이유: 컴포넌트 직접 fetch 금지 원칙 유지하면서 백엔드 없이 로딩→완료 흐름을 그대로 재현. 실제 연동 시 `queryFn`만 교체

#### 순수 로직·마스코트 유틸
- 파일: `frontend/src/features/analysis/lib/{format,timeline,transcript,sparkline,mascot}.ts`
- 변경 내용: `formatTime`(초→M:SS), `segmentAtTime`(재생 위치의 구간), `highlightFillers`(필러 토큰 표시), `buildSparkline`(말속도 SVG path), `scoreToMascot`(점수대별 표정)
- 이유: 비자명 로직을 컴포넌트에서 분리해 가독성·재사용 확보

#### 상단 스티키 오디오·타임라인 플레이어
- 파일: `frontend/src/components/result/AudioTimelinePlayer.tsx`, `TimelineTrack.tsx`
- 변경 내용: `<audio>` + 커스텀 재생 컨트롤, 타임라인을 발화/침묵/필러 색 밴드로 렌더, 플레이헤드 표시, 클릭·키보드(←/→)로 seek
- 이유: "재생과 분석을 같은 화면에서"라는 핵심 인터랙션 충족. 재생 위치는 플레이어 로컬 상태로만 관리(전역 store 불필요)

#### 결과 본문 패널(10종)
- 파일: `frontend/src/components/result/{ReportHeader,TranscriptPanel,ScorePanel,SpeedGraph,SilenceCard,FillerCard,ContentFeedbackPanel,ImprovementList}.tsx`
- 변경 내용: 질문+평가역량, 전사(필러 하이라이트), 종합/세부 점수+마스코트, 말속도 SVG 그래프, 침묵·필러 카드, 내용 피드백(논리/전문성/부합도), 개선 포인트
- 이유: PRD §6.7 결과 화면 항목을 작은 presentational 컴포넌트로 분할(컴포넌트 1파일 패턴 준수)

#### 페이지 조립·라우팅
- 파일: `frontend/src/pages/ResultPage.tsx`, `frontend/src/App.tsx`
- 변경 내용: 10개 컴포넌트를 조립한 결과 페이지, `/result` 라우트 + nav 링크 추가(기존 `/`·`/showcase` 보존)
- 이유: 결과 화면을 독립 라우트로 노출

### 🔧 주요 변경사항 상세

#### 타임라인 동기화
`AudioTimelinePlayer`가 `currentTime`·`duration`을 보유하고 `TimelineTrack`에 전달. 트랙은 각 구간을 길이 비율로 배치하고 `currentTime` 위치에 플레이헤드를 그림. 클릭 시 좌표 비율로 seek.

특이사항:
- 단어 단위 타임스탬프가 없어 전사는 정적 필러 하이라이트만(구간 단위 동기화는 타임라인에서 처리)
- 오디오 메타데이터가 없을 때 타임라인 마지막 구간 끝을 길이로 폴백 → 샘플 음성이 없어도 구간·플레이헤드가 올바른 비율로 표시됨

#### 접근성·디자인 토큰 보완(리뷰 반영)
- 타임라인에 `role="slider"`·`tabIndex`·`aria-value*`·키보드 seek 추가(WCAG 2.1.1)
- 말속도 SVG 색을 `stroke-primary`/`fill-primary` 유틸 클래스로 전환(인라인 색 우회 제거)
- `ReportStatus`를 `AnalysisStatus` 별칭으로 통일(타입 중복 제거)
- 가장 긴 침묵 값 `toFixed(1)` 포맷(실데이터 대비)

### 📦 의존성 변경
- 없음. 기존 react-router-dom·TanStack Query·Tailwind만 사용. 차트 라이브러리·전역 store 미도입(인라인 SVG·로컬 상태로 대체)
