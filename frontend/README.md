# Speech Mirror Frontend

React, TypeScript, Vite, Tailwind CSS, ESLint, Prettier core setup.

## Stack

- React + TypeScript + Vite
- Tailwind CSS via the Vite plugin
- Axios for HTTP requests
- TanStack Query for async server state, loading state, and polling
- Zustand for lightweight client state

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run format
```

## Environment

Copy `.env.example` to `.env.local` and set the backend API URL.
